Ctrl + Cook is a full-stack application designed to help users discover, customize, and save recipes while automatically identifying potential allergens. It integrates with the Spoonacular API for recipe data and uses a custom backend to manage user accounts and personalized cookbooks.

Core Features
Recipe Search & Discovery: Find recipes using the Spoonacular API.

Allergen Highlighting: Automatically identify ingredients that conflict with a user's defined allergies.

Ingredient Substitution: Suggest and allow users to implement ingredient substitutions.

Personal Cookbook: Save customized (or original) recipes to a personal, persistent cookbook.

Authentication: User registration and login (currently simulated via Redux).
